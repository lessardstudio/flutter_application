# Инструкция по компиляции APK для Android

Этот файл описывает процесс сборки установочного файла (`.apk`) для вашего Flutter приложения.

## Предварительные требования
Убедитесь, что у вас установлены:
1. **Flutter SDK**
2. **Android Studio** (с Android SDK и Command-line Tools)
3. **Java** (JDK 17, так как в проекте используется версия 17)

## Шаг 1: Подготовка
Откройте терминал в папке проекта и обновите зависимости:

```bash
flutter pub get
```

## Шаг 2: Сборка APK
Чтобы собрать файл, выполните команду:

```bash
flutter build apk
```

Эта команда соберет `release` версию приложения.
*Примечание:* В данном проекте релизная версия настроена на подпись ключом отладки (debug key), поэтому вы сможете установить этот APK на свой телефон вручную, но не сможете загрузить его в Google Play Store без настройки боевых ключей.

## Шаг 3: Где найти файл
После завершения сборки готовый файл будет находиться по пути:

```
build/app/outputs/flutter-apk/app-release.apk
```

## Установка на устройство
Если ваш телефон подключен к компьютеру (с включенной отладкой по USB), вы можете установить приложение сразу командой:

```bash
flutter install
```

Либо просто скопируйте файл `app-release.apk` на телефон и запустите его.
